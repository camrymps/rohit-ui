Alert

The Alert component displays important messages to users with different severity levels in a Windows 98 style.

Import

import { Alert } from 'rohit-ui';

Examples

Basic Alert
This is a standard alert message.
<Alert>This is a standard alert message.</Alert>
Alert Severity Levels
✓ Success! Your file has been saved.
ℹ Information: Windows 98 was released in 1998.
⚠ Warning: Your system is running low on memory.
✕ Error: A problem has been detected and Windows has been shut down to prevent damage to your computer.
// Success alert <Alert severity="success">Success! Your file has been saved.</Alert> // Information alert <Alert severity="info">Information: Windows 98 was released in 1998.</Alert> // Warning alert <Alert severity="warning">Warning: Your system is running low on memory.</Alert> // Error alert <Alert severity="error">Error: A problem has been detected and Windows has been shut down to prevent damage to your computer.</Alert>
Dismissible Alert
This alert can be dismissed.
<Alert dismissible onDismiss={() => console.log('Alert dismissed')}> This alert can be dismissed. </Alert>
Alert with Title
Alert Title
This is an alert with a title.
<Alert title="Alert Title"> This is an alert with a title. </Alert>
Rohit Mode Alert
ROHIT SAYS: This alert is unnecessarily flashy and takes multiple clicks to dismiss!
<Alert rohitMode> This alert is unnecessarily flashy and takes multiple clicks to dismiss! </Alert>

Note: In actual Rohit Mode, the alert will blink continuously and require multiple dismissal attempts.

Props

Name Type Default Description
children ReactNode - The content of the alert
severity 'success' | 'info' | 'warning' | 'error' 'info' The severity level of the alert, affecting its color and icon
title string - Optional title displayed at the top of the alert
dismissible boolean false Whether the alert can be dismissed
onDismiss function - Callback function called when the alert is dismissed
icon ReactNode Default severity icon Custom icon to display in the alert
rohitMode boolean false Enables Rohit Mode with silly animations and behaviors

CSS Customization

You can customize the Alert component using styled-components:

import styled from 'styled-components'; import { Alert } from 'rohit-ui'; const CustomAlert = styled(Alert)` border: 2px solid purple; background-color: lavender; .alert-title { background-color: purple; color: white; } .alert-icon { color: purple; } `; function MyComponent() { return ( <CustomAlert title="Custom Alert" severity="info"> This alert has custom styling. </CustomAlert> ); }

Rohit Mode Behavior

When Rohit Mode is enabled, the Alert component will:

  • Blink with bright colors to get unnecessary attention
  • Require multiple dismissal attempts before actually closing
  • Occasionally reappear after being dismissed
  • Add "ROHIT SAYS:" prefix to all alert messages
  • Play Windows 98 error sounds at random intervals
