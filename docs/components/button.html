Button

The Button component provides a classic Windows 98-style button with various states and options.

Import

import { Button } from 'rohit-ui';

Examples

Basic Button
Click Me
<Button>Click Me</Button>
Disabled Button
Disabled
<Button disabled>Disabled</Button>
Primary Button
Primary
<Button primary>Primary</Button>
Rohit Mode Button
Rohit Mode
<Button rohitMode>Rohit Mode</Button>

Note: In actual Rohit Mode, the button will spin randomly when hovered and do other unexpected things.

Props

Name Type Default Description
children ReactNode - The content of the button
primary boolean false Makes the button a primary button with themed styling
disabled boolean false Disables the button, preventing interactions
icon string - Icon to display inside the button
iconPosition 'left' | 'right' 'left' Position of the icon relative to the text
size 'small' | 'medium' | 'large' 'medium' Size of the button
fullWidth boolean false Makes the button take up the full width of its container
onClick function - Function to call when the button is clicked
rohitMode boolean false Enables Rohit Mode with silly animations and behaviors

CSS Customization

You can customize the Button component using styled-components:

import styled from 'styled-components'; import { Button } from 'rohit-ui'; const CustomButton = styled(Button)` background-color: #006400; color: white; &:hover { background-color: #008000; } `; function MyComponent() { return ( <CustomButton>Custom Styled Button</CustomButton> ); }

Rohit Mode Behavior

When Rohit Mode is enabled, the Button component will:

  • Spin randomly when hovered
  • Change colors unpredictably
  • Sometimes move away from the cursor when being clicked
  • Occasionally make Windows 98 error sounds
  • Take longer than necessary to perform simple actions

Rohit Mode can be enabled per-component or globally:

// Enable for a single button <Button rohitMode>Unnecessarily Complicated</Button> // Enable globally <RohitUIProvider theme="windows98" globalRohitMode> /* All components will be unnecessarily complicated */ </RohitUIProvider>
