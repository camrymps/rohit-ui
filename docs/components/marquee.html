Marquee

The Marquee component provides a Windows 98-style scrolling text display, perfect for announcements, news tickers, or any content that needs to move across the screen.

Import

import { Marquee } from 'rohit-ui';

Examples

Basic Marquee
Welcome to Rohit UI! This is a basic marquee text that scrolls from right to left. 
<Marquee>
  Welcome to Rohit UI - The unnecessarily complicated Windows 98 style React component library!
</Marquee>
Marquee with Custom Speed
This marquee scrolls faster than the default speed. 
<Marquee speed={10}>
  This marquee is moving faster than the default one!
</Marquee>
Rohit Mode Marquee
✨ Welcome to the MOST AMAZING MARQUEE EVER! ✨ 
<Marquee rohitMode>
  UNNECESSARILY COMPLICATED MARQUEE TEXT THAT WIGGLES AND CHANGES COLOR!
</Marquee>

Note: In Rohit Mode, the marquee will wiggle, change color, and sometimes scroll in random directions.

Props

Name Type Default Description
children ReactNode - Content to display in the marquee
speed number 20 Speed of the marquee animation in seconds
direction 'left' | 'right' | 'up' | 'down' 'left' Direction of the marquee movement
pauseOnHover boolean true Whether to pause the marquee when hovered
className string - CSS class to apply to the marquee container
rohitMode boolean false Enables Rohit Mode with silly animations and behaviors

CSS Customization

You can customize the Marquee component using styled-components:

import styled from 'styled-components';
import { Marquee } from 'rohit-ui';

const CustomMarquee = styled(Marquee)`
  background: linear-gradient(90deg, var(--win98-silver), white);
  border-width: 3px;
  border-style: dashed;
  
  .marquee-content {
    font-family: 'Comic Sans MS', sans-serif;
    text-shadow: 2px 2px var(--win98-navy);
  }
`;

function MyComponent() {
  return (
    
      Custom styled marquee content
    
  );
}

Rohit Mode Behavior

When Rohit Mode is enabled, the Marquee component will:

  • Wiggle or bounce while scrolling
  • Change text color periodically
  • Sometimes scroll in random directions
  • Add random emojis to the text
  • Draw unnecessary attention to itself
