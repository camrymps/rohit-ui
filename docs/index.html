Rohit UI

A Windows 98-inspired React component library with a twist - every component has a special "Rohit Mode" that adds unnecessarily complicated and silly behaviors.

Overview

Rohit UI is a nostalgic journey back to the Windows 98 era, reimagined with modern React and TypeScript. It's perfect for those who want to create retro-styled applications with the convenience of modern development tools.

Every component in Rohit UI comes with two themes:

  • Windows 98 (Default): The classic Windows 98 aesthetic we all remember
  • Rohit Special: A funky variation with bright colors and unique styling

Features

  • 100% React components with TypeScript support
  • Styled with styled-components for easy theme customization
  • Each component includes a "Rohit Mode" easter egg with silly animations or behaviors
  • Full accessibility support
  • Responsive design that works on all devices
  • Lightweight and performant

Component Categories

Rohit UI includes the following component categories:

Basic Components
Button, Checkbox, Radio, TextField, Select, ProgressBar, Window
Advanced Components
Tooltip, Menu, Tabs, Dialog, Accordion, CodeBlock
Additional Components
Alert, Badge, Spinner, Card

The Rohit Experience

Each component includes a "Rohit Mode" that can be activated to add unnecessarily complicated and silly behaviors. For example, buttons might spin, text fields might autocorrect to Rohit-specific language, and code blocks will "Rohit-ify" your code to make it more complex than it needs to be.

It's a fun Easter egg that pays homage to our friend Rohit's unique coding style!
