Themes

Explore the theming capabilities of Rohit UI.

Built-in Themes

Rohit UI comes with two built-in themes:

Windows 98 (Default)

The classic Windows 98 look, featuring:

  • Navy blue title bars
  • Silver gray backgrounds
  • Classic 3D beveled borders
  • Classic sans-serif fonts
  • Pixel-perfect icons and controls
Rohit Special

A funky, colorful variation, featuring:

  • Hot pink title bars
  • Teal backgrounds
  • Exaggerated 3D effects
  • Playful font variations
  • Animated icons and effects

Using Themes

To apply a theme, wrap your application with the RohitUIProvider:

import { RohitUIProvider } from 'rohit-ui'; function App() { return ( <RohitUIProvider theme="windows98"> /* Your app */ </RohitUIProvider> ); }

You can switch themes dynamically:

import React, { useState } from 'react'; import { RohitUIProvider, Button } from 'rohit-ui'; function App() { const [theme, setTheme] = useState('windows98'); const toggleTheme = () => { setTheme(theme === 'windows98' ? 'rohitSpecial' : 'windows98'); }; return ( <RohitUIProvider theme={theme}> <Button onClick={toggleTheme}> Switch to {theme === 'windows98' ? 'Rohit Special' : 'Windows 98'} </Button> /* Rest of your app */ </RohitUIProvider> ); }

Creating Custom Themes

You can create custom themes by extending the base themes:

import { RohitUIProvider, createTheme } from 'rohit-ui'; const customTheme = createTheme({ name: 'myCustomTheme', baseTheme: 'windows98', colors: { primary: '#8b008b', // Dark magenta secondary: '#9acd32', // Yellow green background: '#f0f8ff', // Alice blue text: '#2f4f4f', // Dark slate gray }, fonts: { main: 'Comic Sans MS, cursive', // A very Rohit choice } }); function App() { return ( <RohitUIProvider theme={customTheme}> /* Your app */ </RohitUIProvider> ); }

Theme Properties

The following theme properties can be customized:

Property Type Description
colors.primary string Primary color used for title bars, buttons, etc.
colors.secondary string Secondary color used for accents and highlights
colors.background string Base background color
colors.border string Color used for borders
colors.text string Default text color
fonts.main string Main font family
fonts.heading string Font family for headings
fonts.monospace string Font family for code blocks
shadows.normal string Normal shadow style
shadows.inset string Inset shadow style for pressed buttons, etc.
borders.normal string Normal border style
borders.inset string Inset border style
